# Local citation PDFs (not in git)

Drop source PDFs here for offline reading and `\citehref` links. Filenames
should match bibliography keys when possible (`{bibkey}.pdf`, e.g. `cei224.pdf`,
`lpomsa.pdf`). The book's `sections/citeurls.tex` uses these when present and
falls back to public URLs when they are missing.

This directory is gitignored because many standards and papers are not
redistributable.
